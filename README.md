# Notebook App

A feature-rich Flutter notebook application built using **Riverpod** for state management, **SQLite** for local persistence, and automated data synchronization when an internet connection is available.

---

## Features

* **Complete CRUD Operations:** Create, read, update, and delete notes seamlessly.
* **Swipe to Delete:** Delete notes effortlessly by dragging/swiping a list item.
* **Local Persistence (SQLite):** Your notes are saved locally on the device, ensuring offline capability.
* **Auto Network Sync:** Automatically synchronizes local data with the cloud once an internet connection is detected.
* **State Management:** Powered by **Riverpod** for predictable, clean, and scalable state handling.
* **Dynamic Theming:** Smoothly toggle between **Light Mode** and **Dark Mode**.
